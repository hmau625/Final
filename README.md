# Proyecto de Diseño y Análisis de PC Personalizadas

Este proyecto tiene como objetivo diseñar una **PC personalizada** utilizando datos de hardware y presupuestos de usuarios. A través del análisis de datos, se recomienda la mejor configuración de componentes (procesador, tarjeta gráfica, memoria RAM, almacenamiento, etc.) según las necesidades y el presupuesto proporcionado.

El proyecto sigue la metodología **ASUM-DM**, cubriendo todas las fases desde la comprensión del problema hasta la comunicación de resultados.

## Objetivos
- Proponer configuraciones de PC personalizadas basadas en los requisitos del usuario.
- Analizar qué componentes de hardware son más efectivos según el uso esperado (juegos, trabajo, creación de contenido).
- Construir un modelo para estimar el costo de las configuraciones recomendadas.

## Metodología: ASUM-DM
El proyecto sigue las fases de la metodología **ASUM-DM** para el análisis de datos:

1. **Comprensión del Problema**
2. **Exploración de los Datos (EDA)**
3. **Preparación de los Datos**
4. **Modelado**
5. **Evaluación y Optimización**
6. **Despliegue y Comunicación de Resultados**

## Tecnologías Utilizadas
- **Python** (pandas, numpy, matplotlib, seaborn, scikit-learn)
- **Jupyter Notebook** para análisis interactivo
- **GitHub** para control de versiones
- **Power BI** (opcional, para visualización de resultados)

## Contenido del Repositorio
