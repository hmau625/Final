# Proyecto de Análisis de Datos: Tienda de Electrónica

Este proyecto consiste en un análisis de datos de una **tienda de electrónica** con el objetivo de extraer información relevante para la toma de decisiones comerciales. Se aplica la metodología **ASUM-DM** que cubre todas las fases desde la comprensión del problema hasta la comunicación de resultados.

## Objetivos
- Optimizar ventas y gestión de inventarios.
- Identificar patrones y tendencias en ventas.
- Construir modelos predictivos para análisis de comportamiento de clientes y productos.

## Metodología: ASUM-DM
El proyecto sigue la metodología **ASUM-DM**, que consta de las siguientes fases:

1. **Comprensión del Problema**
2. **Exploración de los Datos (EDA)**
3. **Preparación de los Datos**
4. **Modelado**
5. **Evaluación y Optimización**
6. **Despliegue y Comunicación de Resultados**

## Tecnologías Utilizadas
- **Python** (pandas, numpy, matplotlib, seaborn, scikit-learn)
- **Jupyter Notebook** para análisis interactivo
- **GitHub** para control de versiones
- **Power BI** (opcional, para visualización de resultados)

## Contenido del Repositorio
- **Informe en PDF**: Documentación detallada de cada fase.
- **Notebook**: Código de análisis y modelado.
- **Presentación**: Elevator pitch con los hallazgos clave.
- **Dashboard** (opcional): Visualización interactiva de resultados.

## Ejecución
Clona este repositorio y ejecuta el notebook en tu entorno local para explorar y replicar el análisis de datos.
